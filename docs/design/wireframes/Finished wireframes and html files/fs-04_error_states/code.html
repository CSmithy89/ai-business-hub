FS-04 Error States

A design system reference for various error state components.

Inline Field Error

A standard text input field with a red border, an error icon, and a concise error message.

Please enter a valid email address.

Form Error Banner

A prominent banner at the top of a form summarizing multiple issues with anchor links.

error

Please fix the 3 errors below

  • Full Name is required.
  • A valid Email Address is required.
  • Password must be at least 8 characters.

Card/Section Error

A self-contained card for component-level errors like an API failure.

signal_wifi_off

Could not load data

The connection to the server failed. Please check your network and try again.

Global Banner (Offline)

A full-width banner docked to the top of the viewport to indicate loss of connectivity.

wifi_off You are currently offline. Some features may be unavailable.

Modal (Session Expired)

A modal dialog that appears when a user's session has timed out for security reasons.

lock_clock

Session Expired

For your security, you have been logged out due to inactivity. Please log in again to continue.

Rate Limit Banner

A warning banner that informs the user they have exceeded an action limit and must wait.

timer

Rate limit exceeded

You have made too many requests. Please try again in 45 seconds.

Partial Failure Report

A component for displaying results of a bulk operation, such as an import, with mixed success and failure states.

Import complete with 2 errors

3 of 5 contacts were imported successfully.

    Success
  • cancel contact-02.csv
    View details
  • check_circle contact-03.csv
    Success
  • cancel contact-04.csv
    View details
  • check_circle contact-05.csv
    Success

Full Page Error (500)

A full-page state for critical server-side errors.

error

500

Something went wrong

We're sorry, but our servers are experiencing a problem. Our team has been notified. Please try again later.

Full Page Error (404)

A user-friendly page for when a requested URL cannot be found, with helpful navigation.

search_off

404

Page Not Found

The page you are looking for might have been removed, had its name changed, or is temporarily unavailable.
