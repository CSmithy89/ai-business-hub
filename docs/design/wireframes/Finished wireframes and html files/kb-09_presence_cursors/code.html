Environment Setup Guide

Complete guide to configuring your local development environment for the platform.

1. Prerequisites

Before you begin, ensure you have the following installed on your machine. This setup assumes you are running on a Unix-based system (macOS or Linux). Windows users should use WSL2.

2. Installation Steps

First, clone the repository into your local workspace directory. Bob Use the SSH link provided in your welcome email to authenticate with the git server.

git clone git@github.com:acme-corp/platform-core.git

3. Database Configuration

The platform uses a distributed database architecture. You will need to spin up the local containers using Docker Compose. Make sure your Docker daemon is running with at least 8GB of memory allocated.

Once the containers are up, verify the connection by running the health check script located in the /scripts directory.
