### Key Components

#### 1. Chat Header
- Title: "Ask Your Knowledge Base"
- Scribe agent branding (teal color)
- Connection status indicator (online/offline)
- Chat history toggle

#### 2. Welcome Message
- Scribe introduction
- Suggested questions (based on popular KB topics)
- Quick action buttons:
  - "Find gaps in docs"
  - "Generate summary"
  - "Recent changes"

#### 3. Chat Conversation
- User messages (right-aligned)
- Scribe responses (left-aligned with avatar)
- Message timestamps
- Copy/share message buttons

#### 4. Source Citations
Each AI response includes:
- Source pages used for answer
- Verification status badge (âœ“ Verified / Unverified)
- Confidence score percentage
- Click to navigate to source

#### 5. Response Actions
- **View Full Page**: Opens the primary source page
- **Regenerate**: Re-query with different context
- **Create Page**: Convert answer to new KB page
- **Feedback**: Thumbs up/down for answer quality

#### 6. Input Area
- Multi-line text input
- Send button
- Voice input option
- File attachment (for context)
- @ mention pages for context

### Key Components

#### 1. KB Health Score Card
Summary metrics:
- **Overall Score**: Weighted health percentage
- **Total Pages**: Count with trend
- **Knowledge Gaps**: Detected missing areas
- **Freshness**: Percentage of recently updated content

#### 2. Content Coverage
- Bar chart by documentation area
- Color coding: Green (>80%), Yellow (50-80%), Red (<50%)
- Click-through to see pages in each area
- Coverage goals settings

#### 3. RAG Performance
- **Context Utilization**: How much KB content is used in AI responses
- **Top Retrieved Pages**: Most frequently used in RAG queries
- **Query success rate**: Questions answered vs. failed
- **Average confidence score**: AI answer quality metric

#### 4. Stale Content
- List of pages needing review
- Age indicator (months since update)
- Quick actions: Review, Archive, Assign
- Configurable staleness threshold (default: 90 days)

#### 5. Usage Patterns
- Page view trends over time
- Most active contributors
- Popular search queries
- Time-of-day usage patterns

#### 6. Knowledge Gaps Panel
- AI-detected documentation gaps
- Gap categories (Security, Testing, etc.)
- Gap count per category
- Actions:
  - Generate draft documentation
  - Notify content owners
  - Export gap report
