### Key Components

1. **Search & Filters Bar**
   - Full-text search
   - Filter dropdown (industry, size, owner, health)
   - View toggle (table/card)
   - Add company button

2. **Quick Filter Chips**
   - Segment filters (Enterprise, Mid-Market, SMB)
   - Status filters (Customer, Prospect)
   - Counts per filter

3. **Company Table/Grid**
   - Checkbox selection
   - Company name with domain
   - Industry and size
   - Contact count
   - Health score with color indicator

4. **Bulk Actions**
   - Assign owner
   - Add tags
   - Export selected
   - Delete (with confirmation)

5. **Atlas Agent Suggestion**
   - Data enrichment prompt
   - Quick action link
