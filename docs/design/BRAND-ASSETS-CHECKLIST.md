## Tools & Resources

### Design Tools
- **Figma** - Primary design tool
- **Adobe Illustrator** - Vector illustration
- **Adobe After Effects** - Animation source
- **LottieFiles** - Animation hosting

### Export Guidelines
- **SVG**: Optimized with SVGO
- **PNG**: 2x and 3x retina versions
- **Lottie**: Under 50KB per animation
- **PDF**: CMYK for print, RGB for digital

### File Naming Convention
```
{brand}-{asset-type}-{variant}-{size}.{format}

Examples:
hyvve-logo-primary.svg
hyvve-logomark-64.png
hub-celebrating-256.png
social-banner-twitter.png
```
