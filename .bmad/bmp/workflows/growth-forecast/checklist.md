# Growth Forecast Validation Checklist

## Growth Model
- [ ] Growth model selected and justified
- [ ] Model appropriate for business type
- [ ] Model assumptions documented

## Growth Levers
- [ ] Primary growth levers identified
- [ ] AARRR framework applied
- [ ] Target metrics defined
- [ ] Strategies aligned with resources

## Projections
- [ ] Monthly Year 1 projections
- [ ] Quarterly Year 2-3 projections
- [ ] Annual Year 4-5 projections
- [ ] New vs churned customers tracked
- [ ] Growth rates calculated

## Scenarios
- [ ] Conservative scenario provided
- [ ] Realistic scenario provided
- [ ] Optimistic scenario provided
- [ ] Scenario assumptions documented
- [ ] Scenarios are differentiated

## Milestones
- [ ] Key milestones defined
- [ ] Target dates realistic
- [ ] Milestones align with growth model

## Risk Assessment
- [ ] Growth risks identified
- [ ] Mitigations documented
- [ ] Key assumptions flagged

## Anti-Hallucination
- [ ] Ranges used where uncertain
- [ ] Assumptions explicit
- [ ] Comparable benchmarks cited
- [ ] Uncertainty acknowledged
