# Pitch Deck Validation Checklist

## Overall Structure
- [ ] 10-12 slides total
- [ ] Each slide has one key message
- [ ] Logical flow/narrative
- [ ] Speaker notes included

## Title Slide
- [ ] Company name clear
- [ ] Tagline memorable
- [ ] One-liner compelling

## Problem Slide
- [ ] Problem clearly articulated
- [ ] Target customer identified
- [ ] Pain level conveyed
- [ ] Story or example included

## Solution Slide
- [ ] Solution clearly explained
- [ ] Differentiator highlighted
- [ ] Demo/visual suggested
- [ ] Avoided feature list

## Market Slide
- [ ] TAM/SAM/SOM included
- [ ] Sources cited
- [ ] Growth rate shown
- [ ] Market trend noted

## Business Model Slide
- [ ] Revenue model clear
- [ ] Pricing mentioned
- [ ] Unit economics shown

## Traction Slide
- [ ] Relevant metrics shown
- [ ] Growth highlighted
- [ ] Milestones listed
- [ ] (Or validation signals if pre-revenue)

## Competition Slide
- [ ] Competitors acknowledged
- [ ] Differentiation clear
- [ ] Defensibility addressed
- [ ] Didn't say "no competition"

## Team Slide
- [ ] Key members introduced
- [ ] Relevant experience shown
- [ ] Team advantage stated

## Financials Slide
- [ ] Projections included
- [ ] Path to profit shown
- [ ] Key assumptions clear

## Ask Slide
- [ ] Amount clearly stated
- [ ] Use of funds specified
- [ ] Milestones defined
- [ ] Contact info included

## Quality Check
- [ ] Numbers consistent throughout
- [ ] No typos
- [ ] Professional tone
- [ ] Can be presented in ~15 minutes
