# Visual Identity Validation Checklist

## Input Validation

- [ ] Brand archetype loaded from strategy workflow
- [ ] Target personas available for visual preferences
- [ ] Positioning statement received
- [ ] Competitive visual landscape reviewed
- [ ] Existing visual elements documented (if any)

## Logo System

### Logo Specifications:

- [ ] Logo type selected with rationale
- [ ] Primary logo defined
- [ ] Secondary logo defined
- [ ] Icon/mark version defined
- [ ] Horizontal lockup specified
- [ ] Vertical lockup specified
- [ ] Single-color (dark) version
- [ ] Single-color (light) version

### Usage Rules:

- [ ] Clear space rules defined (with multiplier)
- [ ] Minimum print size specified
- [ ] Minimum digital size specified
- [ ] Logo don'ts documented (8+ items)
- [ ] Background usage guidelines

### File Specifications:

- [ ] Vector formats listed (AI, EPS, SVG, PDF)
- [ ] Raster formats listed (PNG, JPG)
- [ ] Size variations specified
- [ ] Naming convention defined

## Color Palette

### Primary Colors:

- [ ] Primary color defined
- [ ] Secondary color defined
- [ ] Accent color defined
- [ ] All colors have HEX codes
- [ ] All colors have RGB values
- [ ] All colors have CMYK values
- [ ] All colors have Pantone codes
- [ ] Color ratios defined (60-30-10)

### Extended Palette:

- [ ] White defined
- [ ] Light gray defined
- [ ] Medium gray defined
- [ ] Dark gray defined
- [ ] Black defined
- [ ] Neutrals have specifications

### Semantic Colors:

- [ ] Success color defined
- [ ] Warning color defined
- [ ] Error color defined
- [ ] Info color defined

### Tints & Shades:

- [ ] Primary color tints (10%, 25%, 50%)
- [ ] Primary color shades (75%, 90%)
- [ ] Secondary tints/shades (if needed)

### Accessibility:

- [ ] Contrast ratios calculated
- [ ] WCAG compliance verified (AA minimum)
- [ ] Accessible combinations documented

## Typography

### Font Selection:

- [ ] Primary font selected
- [ ] Primary font weights specified
- [ ] Primary font source/license noted
- [ ] Secondary font selected
- [ ] Secondary font weights specified
- [ ] Monospace font selected (if needed)
- [ ] All fonts have fallback stacks

### Type Scale:

- [ ] Base size defined
- [ ] Scale ratio selected
- [ ] All heading sizes specified (H1-H6)
- [ ] Body sizes specified
- [ ] Caption/small sizes specified
- [ ] Line heights defined
- [ ] Letter spacing defined
- [ ] Font weights assigned per level

### Typography Guidelines:

- [ ] Paragraph spacing defined
- [ ] Maximum line length specified
- [ ] Weight usage guidelines

## Imagery

### Photography:

- [ ] Color treatment defined
- [ ] Lighting style specified
- [ ] Composition guidelines
- [ ] Subject focus direction
- [ ] Mood/emotion guidance
- [ ] Do's list (4+ items)
- [ ] Don'ts list (4+ items)

### Illustration:

- [ ] Style type defined (if applicable)
- [ ] Color approach specified
- [ ] Detail level guidelines

### Iconography:

- [ ] Icon style specified
- [ ] Stroke weight defined
- [ ] Corner radius defined
- [ ] Grid system specified
- [ ] Required icon categories listed

## Visual Elements

### Patterns:

- [ ] Pattern style defined
- [ ] Pattern elements specified
- [ ] Pattern color usage
- [ ] Pattern usage guidelines

### Graphic Devices:

- [ ] Primary shape defined
- [ ] Divider style specified
- [ ] Container specifications

### Layout:

- [ ] Grid system defined
- [ ] Column count specified
- [ ] Gutter width defined
- [ ] Margin specification
- [ ] Spacing scale defined

## Consistency Validation

- [ ] Visual direction aligns with archetype
- [ ] Colors support brand personality
- [ ] Typography matches brand voice
- [ ] All elements work together
- [ ] System is flexible for applications
- [ ] Accessibility requirements met

## Document Quality

- [ ] All sections complete
- [ ] No placeholder text
- [ ] Color codes are accurate
- [ ] Font names are correct
- [ ] Specifications are precise
- [ ] Quick reference included
- [ ] Saved to correct path
