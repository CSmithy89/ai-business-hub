# Brand Strategy Validation Checklist

## Input Validation

- [ ] Business name confirmed
- [ ] Business description from BMP loaded
- [ ] Target audience definition available
- [ ] Value proposition statement received
- [ ] Competitive landscape identified
- [ ] Growth aspirations documented

## Competitive Analysis

- [ ] 3-5 direct competitors identified
- [ ] Competitor archetypes assessed
- [ ] Competitor positioning analyzed
- [ ] Perceptual map created
- [ ] White space opportunities identified
- [ ] Differentiation strategy defined
- [ ] Analysis based on real data (not invented)

## Archetype Selection

- [ ] All 12 archetypes considered
- [ ] Scoring criteria applied consistently
- [ ] Primary archetype selected
- [ ] Secondary archetype identified (if applicable)
- [ ] Selection rationale documented
- [ ] Archetype aligns with audience expectations
- [ ] Archetype differentiates from competitors
- [ ] User confirmed archetype selection

## Target Personas

### For Each Persona:

- [ ] Demographics complete (age, location, income, education, occupation)
- [ ] Psychographics defined (values, attitudes, interests, lifestyle)
- [ ] Pain points identified (3+)
- [ ] Goals documented (3+)
- [ ] Brand expectations articulated
- [ ] Representative quote included
- [ ] Based on provided audience data (not invented)

### Persona Portfolio:

- [ ] 2-3 personas created
- [ ] Primary persona identified
- [ ] Personas cover key audience segments
- [ ] Communication preferences defined

## Positioning Statement

- [ ] Target audience clearly defined
- [ ] Need/problem articulated
- [ ] Category identified
- [ ] Key benefit stated
- [ ] Competitors named
- [ ] Key differentiator explained
- [ ] Statement is clear and compelling
- [ ] Differentiation is defensible

## Tagline Development

- [ ] 3-5 options developed
- [ ] Options evaluated for strengths/concerns
- [ ] Recommended tagline selected
- [ ] Rationale documented
- [ ] Tagline is short (2-4 words)
- [ ] Tagline is memorable
- [ ] Tagline captures brand essence
- [ ] Tagline is timeless (not trendy)

## Messaging Framework

- [ ] Brand promise defined (one line)
- [ ] 3-5 key messages created
- [ ] Each message has proof points
- [ ] Use cases for each message defined
- [ ] Elevator pitch developed (30 seconds)
- [ ] Boilerplate written (50-100 words)
- [ ] Messages support value proposition
- [ ] Messages align with archetype voice

## Strategy Consistency

- [ ] Archetype aligns with positioning
- [ ] Personas reflect BMP target audience
- [ ] Messaging supports value proposition
- [ ] Differentiation is clear
- [ ] All elements feel cohesive
- [ ] Strategy is actionable

## Document Quality

- [ ] All sections completed
- [ ] No placeholder text remains
- [ ] All tables populated
- [ ] Formatting is consistent
- [ ] Spelling and grammar checked
- [ ] Document saved to correct path

## Output Artifacts

- [ ] brand-strategy-document.md complete
- [ ] archetype-profile.md generated
- [ ] persona-cards.md generated
- [ ] positioning-statement.md generated
- [ ] messaging-framework.md generated
- [ ] competitive-analysis.md generated
