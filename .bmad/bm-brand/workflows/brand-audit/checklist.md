# Brand Audit Validation Checklist

## Input Validation

- [ ] Brand guidelines loaded
- [ ] Touchpoint samples collected
- [ ] Competitor information gathered
- [ ] Performance data collected (if available)

## Visual Consistency Audit

- [ ] Logo usage checked across touchpoints
- [ ] Clear space compliance verified
- [ ] Minimum size compliance verified
- [ ] Color accuracy checked (HEX/RGB/CMYK)
- [ ] Typography compliance verified
- [ ] Imagery style consistency checked
- [ ] Specific issues documented

## Verbal Consistency Audit

- [ ] Voice attributes assessed
- [ ] Tone appropriateness evaluated
- [ ] Messaging consistency checked
- [ ] Tagline usage verified
- [ ] Key messages assessed
- [ ] Writing guidelines compliance checked
- [ ] Specific issues documented

## Touchpoint Analysis

- [ ] All touchpoints scored
- [ ] Visual scores per touchpoint
- [ ] Verbal scores per touchpoint
- [ ] Issues documented per touchpoint
- [ ] Best performers identified
- [ ] Improvement needs identified

## Competitive Analysis

- [ ] 3-5 competitors analyzed
- [ ] Comparison factors scored
- [ ] Perceptual position mapped
- [ ] Strengths identified
- [ ] Gaps identified

## Health Scoring

- [ ] Visual consistency score calculated
- [ ] Verbal consistency score calculated
- [ ] Touchpoint quality score calculated
- [ ] Competitive position score calculated
- [ ] Overall score calculated
- [ ] Weighted scoring applied
- [ ] Score interpretation provided

## Recommendations

- [ ] Immediate actions (30-day) defined
- [ ] Short-term actions (90-day) defined
- [ ] Long-term actions (12-month) defined
- [ ] Actions prioritized
- [ ] Impact assessed per action

## Report Quality

- [ ] Executive summary complete
- [ ] All sections populated
- [ ] Evidence cited for findings
- [ ] Scores justified
- [ ] No placeholder text
- [ ] Next steps defined

## Output Artifacts

- [ ] brand-audit-report.md complete
- [ ] health-scorecard.md generated
- [ ] consistency-checklist.md generated
- [ ] action-plan.md generated
