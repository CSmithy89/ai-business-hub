# Brand Voice Validation Checklist

## Input Validation

- [ ] Brand archetype loaded from strategy workflow
- [ ] Target personas available
- [ ] Positioning statement received
- [ ] Messaging framework accessible
- [ ] Previous voice elements reviewed (if existing)

## Voice Attributes

### For Each Attribute (3-4 required):

- [ ] Clear definition provided
- [ ] Rationale connects to archetype
- [ ] "This means..." examples included
- [ ] "This doesn't mean..." boundaries set
- [ ] Before/after examples demonstrate application
- [ ] Attribute is actionable for writers

### Attribute Portfolio:

- [ ] 3-4 attributes defined
- [ ] Attributes complement each other
- [ ] No contradicting attributes
- [ ] Attributes derive from archetype
- [ ] Attributes differentiate from competitors

## Tone Spectrum

### For Each Tone Context:

- [ ] Clear "when to use" guidance
- [ ] Voice attribute emphasis defined
- [ ] Specific characteristics listed
- [ ] Example copy provided
- [ ] Tone is distinguishable from others

### Coverage:

- [ ] Celebratory tone defined
- [ ] Educational tone defined
- [ ] Supportive tone defined
- [ ] Urgent tone defined
- [ ] Conversational tone defined
- [ ] Apologetic tone defined
- [ ] All relevant contexts covered

- [ ] Examples for each

### Capitalization:

- [ ] Headline style specified

### Numbers & Dates:

- [ ] Time format

### Formatting:

- [ ] Contraction policy

- [ ] Abbreviation rules
