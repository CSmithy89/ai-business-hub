# Brand Voice Development Workflow Instructions

<critical>Voice must derive from archetype - no arbitrary personality choices</critical>
<critical>Tone spectrum must cover all actual communication contexts</critical>
<critical>All guidelines must include specific examples - no vague directions</critical>

## Purpose

This workflow develops the verbal identity system for the brand. It defines HOW the brand communicates through voice attributes, tone variations, writing guidelines, and vocabulary standards. All content creation depends on this foundation.

Standard contexts to address:

| Context | Typical Situation | Tone Direction |
|---------|------------------|----------------|
| **Celebratory** | Wins, launches, milestones | Excited, proud |
| **Educational** | How-tos, tutorials, guides | Clear, helpful |
| **Supportive** | Customer service, troubleshooting | Patient, empathetic |
| **Urgent** | Alerts, deadlines, critical | Direct, clear |
| **Conversational** | Social media, casual | Light, engaging |
| **Formal** | Legal, compliance, contracts | Professional, precise |
| **Sales** | Pitches, proposals, marketing | Persuasive, confident |
| **Apologetic** | Errors, delays, issues | Sincere, accountable |

## Anti-Hallucination Protocol

- Voice attributes must derive from archetype selection
- Tone calibrations must serve real communication needs
- All examples must be plausible for the brand
- Guidelines must be specific and actionable
- Do not invent content samples without context
