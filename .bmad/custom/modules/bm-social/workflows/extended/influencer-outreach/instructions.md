# Influencer Outreach Workflow

## Purpose
Manage influencer relationships, from identification through collaboration and performance tracking.

## Context Variables
- `{{influencer_handle}}` - Target influencer identifier
- `{{platform}}` - Primary platform for influencer
- `{{campaign_id}}` - Optional associated campaign
- `{{outreach_type}}` - Type of collaboration sought

## Prerequisites
- Social listening enabled
- CRM integration (optional)
- Outreach templates prepared
- Budget allocated (if paid collaboration)

## Outreach Templates

| Template | Use Case |
|----------|----------|
| Cold Intro | First contact, no prior relationship |
| Fan Follow-up | They've mentioned brand positively |
| Collaboration Pitch | Specific campaign invitation |
| Event Invite | Conference/launch event |
| Affiliate Offer | Ongoing partnership program |

## Influencer Tiers

| Tier | Followers | Typical Rate | Approach |
|------|-----------|--------------|----------|
| Nano | 1K-10K | Product only | Direct DM |
| Micro | 10K-100K | $100-$1K | Personalized pitch |
| Mid | 100K-500K | $1K-$10K | Formal proposal |
| Macro | 500K-1M | $10K-$50K | Manager/agency |
| Mega | 1M+ | $50K+ | Agency required |

## Error Handling

| Error | Resolution |
|-------|------------|
| No response | Follow up at 7 days |
| Declined | Thank them, add to nurture list |
| Pricing mismatch | Counter or explore alternatives |
| Content issues | Provide clearer brief |

## Events Emitted
- `social.influencer.identified` - New prospect found
- `social.influencer.contacted` - Outreach sent
- `social.influencer.responded` - Reply received
- `social.influencer.collaborated` - Content published
