# Customer Discovery Validation Checklist

## Customer Segment Definition

- [ ] Primary target segment clearly defined
- [ ] Segment size estimated with sources
- [ ] Demographics/firmographics documented
- [ ] Psychographics captured
- [ ] Segment prioritization justified

## Jobs-to-be-Done

- [ ] Functional jobs identified
- [ ] Emotional jobs identified
- [ ] Social jobs identified
- [ ] Jobs ranked by importance
- [ ] Current solution satisfaction assessed

## Pain Point Validation

- [ ] Pain points sourced from research
- [ ] Frequency documented
- [ ] Severity assessed
- [ ] Current workarounds identified
- [ ] Cost of problem quantified

## Persona Quality

- [ ] 2-3 personas created
- [ ] Personas based on real data
- [ ] Each persona has distinct needs
- [ ] Decision-making authority noted
- [ ] Representative quotes included

## Buying Process

- [ ] Buying stages mapped
- [ ] Key touchpoints identified
- [ ] Influencers and decision-makers noted
- [ ] Typical timeline understood

## Economics

- [ ] Willingness to pay researched
- [ ] Current spending documented
- [ ] ROI potential calculated
- [ ] LTV estimated
- [ ] CAC projections made

## Anti-Hallucination

- [ ] No invented customer quotes
- [ ] All statistics have sources
- [ ] Personas based on research, not fiction
- [ ] Assumptions clearly labeled
